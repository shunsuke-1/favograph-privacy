# FavoGraph Support

FavoGraph は、撮影したい写真やお気に入りの写真を **シーンごとに整理・管理できるアプリ**です。

カメラマンの撮影準備やロケーション調査、  
写真のアイデア管理、思い出の整理など、さまざまな用途で利用できます。

---

# 主な機能

### シーン管理
撮影したいテーマや場所ごとにシーンを作成できます。

例

- 桜
- 夜景
- カフェ
- 旅行
- ポートレート

---

### 写真の保存
撮りたい写真の参考画像を保存できます。

例えば

- SNSで見つけた写真
- 撮影の構図メモ
- ロケーションの参考写真
- 撮影アイデア

---

### タグ管理
写真やシーンにタグを付けて整理できます。

例

- 春
- 桜
- 夕焼け
- 海
- ストリート

---

### シンプルな写真整理
シンプルなUIで、写真を見やすく管理できます。

---

# プレミアム機能

無料版では作成できるシーン数に制限があります。

プレミアム版では

- シーン数の制限解除

が可能になります。

※ 買い切り型のアプリ内課金です。

# よくある質問

### Q. 写真はどこに保存されますか？

写真やシーンの情報は **ユーザーの端末内に保存されます**。  
外部サーバーにアップロードされることはありません。

---

### Q. 写真は他人に共有されますか？

共有されません。  
すべてのデータは端末内で管理されます。

---

### Q. 課金情報は保存されますか？

課金処理は **Apple の App Store の仕組み**によって管理されます。  
アプリの開発者が決済情報を取得することはありません。
